## 🏥 Descripción

Proyecto desarrollado en **Odoo** para la gestión de una clínica dental.  
Permite administrar citas, pacientes y recetas médicas de manera eficiente.

Además, incluye una **API REST basada en autenticación**, que permite:
- Iniciar sesión en el sistema  
- Crear nuevas citas de forma segura  

---

## 🚀 Características

- Gestión de pacientes  
- Administración de citas  
- Registro de prescripciones médicas  
- API REST con autenticación  
- Creación de citas mediante API  

---

## 🛠️ Tecnologías Utilizadas

- Odoo (ERP)  
- API REST  
- Python  
- JavaScript  
- SCSS  

---

## 🔐 Funcionalidad de la API

- Autenticación de usuarios  
- Creación de citas mediante endpoints  
- Integración con el sistema principal  

---

## 📌 Uso del Sistema

Este sistema está diseñado para:
- Clínicas dentales 🦷  
- Consultorios odontológicos  
- Gestión digital de pacientes y tratamientos
